## Part 01 — Node.js & npm Basics

- Node.js = Chrome's V8 engine outside the browser
- Ryan Dahl wrapped it in C++
- `npm init -y` → project initialize
- `npm i <package>` → install package
- All packages go into `node_modules/`
- `node_modules` deleted? → `npm install` to restore

### Key Commands
\```bash
npm init -y
npm i slugify
npm install
\```
